# Getting Avg Speed of Cars recorded in a particular RSUID in "m/s"
#we will pull mean of speeds recorded by a particular RSUID
ASpeed = []
for i in list(range(0,len(RSUID))):
     ASpeed.append(df[df["A"] == RSUID[i]]["I"].mean())

We have created another Table with 3 columns

Column 1 (Road_ID) : Represents unique RSUID present in the data

Column 2 (Total_No_of_Cars) : Represents the total no of cars passing through a certain RSUID

Column 3 (Avg_Speed) : Represents the avg speed in m/s recorded for a particular RSUID for all the cars passing through the device
